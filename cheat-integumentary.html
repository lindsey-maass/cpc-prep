Section Overview

The Integumentary System codes (10000–19999) cover skin, subcutaneous tissue, nails, and breasts. Procedures range from simple biopsies to complex skin grafts. This section is heavily tested on size, location, and repair type distinctions.

Critical Coding Tips

Size Always Matters

  • Lesion excision: excised diameter = lesion + margins
  • Wound repair: code by total length in centimeters
  • Burns: code by percentage of body surface area
  • Always measure in centimeters -- know the size before coding

Location Groups

  • Simple locations: Trunk, arms, legs
  • Complex locations: Face, ears, eyelids, nose, lips, mucous membranes, hands, feet, genitalia
  • Location determines which code range to use for excisions and repairs

Wound Repair Types

  • Simple (12001–12021): One-layer closure, minimal contamination
  • Intermediate (12031–12057): Layered closure or contaminated wound
  • Complex (13100–13160): Scar revision, debridement, or retention sutures

Multiple Repairs Rule

  • Add lengths together if same type AND same anatomic group
  • Code separately if different repair types or different anatomic groups
  • List most complex/highest-paying code first

Destruction vs. Excision

  • Destruction (17000–17999): Tissue destroyed, not removed (laser, freezing, chemicals)
  • Excision (11400–11646): Tissue cut out and removed, usually for pathology

Lesion Removal Rules

  • Each separate lesion gets its own code
  • Benign vs. malignant = different code ranges (11400s vs. 11600s)
  • Code by lesion size AND location AND whether benign or malignant

Common Exam Traps

  • Do not confuse excision with destruction -- destruction has no pathology specimen
  • Do not add wound lengths from different repair types or different body areas
  • Do not forget that excised diameter includes margins, not just the lesion size
  • Do not use the same code range for benign and malignant lesions
  • Do not overlook that complex repairs include more than just layered closure

CPC Exam Tips

  • Tab each subsection: biopsies, excisions, repairs, destruction, skin grafts, breast
  • Write the anatomic location groupings at the top of the repairs section
  • Practice calculating excised diameters -- these questions are fast if you know the formula
  • Know that simple repair code descriptors specify "superficial" and "single layer"
