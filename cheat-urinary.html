Section Overview

The Urinary System (50010–53899) covers the kidneys, ureters, bladder, and urethra. Key concepts include the "separate procedure" designation for cystoscopy, endoscopic hierarchy, approach-based coding, and specific rules for stone removal procedures.

Critical Rule: Many urinary codes are designated as "separate procedures." This means they are typically included in more extensive procedures performed at the same site during the same session. Do not code a "separate procedure" when it is performed as part of a larger urinary procedure.

Critical Coding Tips

Kidney Procedures

  • Code by approach: open, laparoscopic, or percutaneous
  • Nephrectomy: radical vs. partial vs. simple = different codes
  • Percutaneous nephrostomy: with or without imaging guidance
  • Stone removal: specify approach (open, percutaneous, ureteroscopic, shockwave)

Cystoscopy Rules

  • Diagnostic cystoscopy is included in surgical cystoscopy -- do not code separately
  • Code additional procedures performed during cystoscopy with add-on codes
  • TURBT (52234–52240): code by size of lesion removed
  • Cystoscopy with stent placement: code stent in addition to cystoscopy

Stone Removal (Calculus)

  • ESWL (lithotripsy): 50590 -- does not require endoscopy
  • Ureteroscopy with stone removal: approach-specific codes
  • Open ureterolithotomy: code by location (upper, middle, lower ureter)
  • Basket extraction vs. laser lithotripsy = different approaches

Urodynamics

  • Urodynamic tests (51725–51797) have professional and technical components
  • Code each test separately when multiple urodynamic studies are performed
  • Uroflowmetry (51736) is frequently billed as a standalone service

Common Exam Traps

  • Do not code diagnostic cystoscopy when surgical cystoscopy is performed at the same session
  • Do not ignore "separate procedure" designations in the code descriptors
  • Do not confuse nephrectomy types: radical, partial, and simple have different indications and codes
  • Do not code ESWL with cystoscopy unless the cystoscopy is a separately identifiable service
  • Do not miss bilateral indicators in ureteral procedures -- modifier -50 may apply

CPC Exam Tips

  • Write "Separate Procedure = included in more extensive urinary procedures" at the top of the bladder section
  • Tab kidney, ureter, bladder, and urethra sections separately
  • Know ESWL code 50590 -- it appears frequently in exam questions
  • TURBT size thresholds matter -- write them near 52234
