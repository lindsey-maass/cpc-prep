Section Overview

The Digestive System (40490–49999) covers the entire GI tract from lips to anus, plus accessory organs including liver, gallbladder, and pancreas. The section follows anatomical organization from top to bottom and is heavily tested on endoscopy coding rules.

Key Rule: Diagnostic endoscopy is always included in surgical endoscopy. Never code both in the same session for the same anatomical level. Always code the most extensive procedure performed.

Critical Coding Tips

Upper GI Endoscopy (EGD)

  • EGD codes (43235–43259): esophagus through duodenum
  • Colonoscopy codes require scope to reach cecum or beyond
  • Sigmoidoscopy codes: scope to splenic flexure or less
  • Code by the furthest extent of the scope

Colonoscopy Screening vs. Diagnostic

  • Screening colonoscopy: no symptoms, routine preventive
  • Diagnostic colonoscopy: performed due to symptoms or abnormal findings
  • If screening converts to diagnostic (polyp removed), coding changes
  • Know the difference for both CPT and ICD-10-CM coding

Hernia Repairs

  • Code by hernia type: inguinal, femoral, umbilical, incisional, etc.
  • Initial vs. recurrent = different codes
  • Reducible vs. incarcerated vs. strangulated = different codes
  • Patient age matters for some hernia codes (under 5 vs. 5+)

Laparoscopic vs. Open

  • Laparoscopic and open procedures have separate code ranges
  • If laparoscopic converts to open, code only the open procedure
  • Do not code both laparoscopic and open for the same procedure

Common Exam Traps

  • Do not code diagnostic EGD separately when surgical EGD is performed
  • Do not use sigmoidoscopy codes when the scope reaches the cecum (use colonoscopy)
  • Do not forget that hernia coding requires knowing type, initial vs. recurrent, and reducibility
  • Do not code both laparoscopic and open when laparoscopic converts to open
  • Do not overlook age-based hernia codes for pediatric patients

CPC Exam Tips

  • Write the endoscopy hierarchy rule at the top of the endoscopy code ranges
  • Tab each major organ group for fast navigation
  • Write "initial vs. recurrent + reducible vs. incarcerated" near hernia codes
  • Know that ERCP (43260–43278) codes include both the endoscopy and the fluoroscopic guidance
