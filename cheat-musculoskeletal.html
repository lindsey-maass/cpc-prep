Section Overview

The Musculoskeletal System (20000–29999) is the LARGEST section in CPT and includes bones, joints, muscles, tendons, ligaments, and cartilage. Codes are organized anatomically from head to toe. This section is heavily tested on fracture care terminology and treatment type distinctions.

Fracture Care: Critical Terminology

Treatment Types

  • Closed treatment: No surgical opening to fracture site (may or may not include manipulation)
  • Open treatment: Surgical opening to fracture site for reduction or internal fixation
  • Percutaneous: Fracture fragments fixed through skin without opening the site

Reduction Types

  • Without manipulation: No manual attempt to realign fragments
  • With manipulation: Manual realignment of bone fragments
  • Closed treatment WITHOUT manipulation = immobilization only

Cast Application Rules

  • Cast applied as part of fracture care is included -- do not code separately
  • Cast applied by a different physician = code separately (29000–29799)
  • Cast replacement = separate code regardless of who applied original

Joint Injections / Aspirations

  • Code by joint (small, intermediate, large) not by substance injected
  • Small joints: fingers, toes
  • Intermediate: temporomandibular, wrist, elbow, ankle
  • Large: shoulder, hip, knee

Common Exam Traps

  • Do not confuse "open fracture" (the injury) with "open treatment" (the procedure) -- they are not the same
  • Do not code cast application separately when it is part of the fracture care package
  • Do not code diagnostic arthroscopy when surgical arthroscopy is performed at the same session
  • Do not code manipulation under anesthesia with the same joint's surgical procedure
  • Do not overlook the "General" codes (20000s) -- they apply to any anatomical site

CPC Exam Tips

  • Tab each major body area so you can flip quickly during the exam
  • Write the three treatment types (closed/open/percutaneous) near the first fracture code range you see
  • Know that arthroscopy codes are ALWAYS more specific than open procedure codes when available
  • This is the largest section -- budget extra study time here
