Section Overview

The cardiovascular section is one of the most complex in CPT. It requires strong anatomical knowledge, understanding of professional vs. technical component coding, and strict attention to bundling rules. This section covers pacemakers, open heart surgery, cardiac catheterization, interventional cardiology, and peripheral vascular surgery.

Key Concept: Many cardiovascular procedures involve both a professional component (physician work) and a technical component (equipment/facility). Know when to code globally vs. with component modifiers (-26 and -TC).

Critical Coding Tips

Pacemakers / ICDs

  • Code by system type: single chamber vs. dual chamber vs. biventricular
  • Code separately for: generator, lead(s), and pocket creation/revision
  • Upgrade codes exist -- do not code removal + insertion when an upgrade code applies
  • Revision and replacement codes differ from initial implant codes

CABG (Coronary Artery Bypass Grafting)

  • Code by graft type: arterial, venous, or combined
  • Add-on codes for additional grafts beyond the first
  • Arterial grafts: internal mammary, radial artery, etc.
  • Procurement of saphenous vein graft coded separately when applicable

Cardiac Catheterization

  • Right heart cath, left heart cath, and combined cath = different codes
  • LV angiography is typically included in left heart catheterization
  • Diagnostic angiography is usually bundled when intervention is performed on the same vessel
  • Combined right + left heart cath has its own code -- do not code separately

Vascular Families (Selective Catheterization)

  • Code the most selective catheterization position reached in each vascular family
  • First order = main vessel off aorta
  • Second order = branch off first order vessel
  • Third order = branch off second order vessel
  • Code each vascular family separately

Central Venous Access Devices

FactorDetermines Code Selection
Patient ageUnder 5 years vs. 5 years and older
Device typeNon-tunneled vs. tunneled vs. implanted port
Access siteCentral vs. peripheral insertion (PICC)
Service typeInsertion vs. repair vs. partial vs. complete removal

Common Exam Traps

  • Do not code diagnostic angiography separately when an interventional procedure is performed on the same vessel in the same session
  • Do not code balloon angioplasty separately when a stent is placed in the same vessel (stent includes angioplasty)
  • Do not use modifier -51 on add-on codes (they are modifier -51 exempt)
  • Do not code pacemaker removal + insertion when an upgrade code exists
  • Do not code each individual stent in the same vessel separately
  • Do not miss the age cutoffs in central venous access codes

CPC Exam Tips

  • Tab major subsections: pacemaker, CABG, cardiac cath, PCI, central access, vascular
  • Write the vascular family order (1st/2nd/3rd) near selective catheterization codes
  • Draw a simple coronary anatomy diagram (LAD, LCX, RCA) in your book margin
  • This section rewards heavy study time -- it is complex and heavily tested
