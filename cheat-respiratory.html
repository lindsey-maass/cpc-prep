Section Overview

This section covers the respiratory tract from the nose to the lungs (30000–32999), plus the hemic and lymphatic system (38100–38999) and the mediastinum and diaphragm (39000–39599). Endoscopy hierarchy and unilateral/bilateral rules are the most heavily tested concepts here.

Endoscopy Hierarchy Rule

Key Rule: Always code to the highest level of endoscopy performed. If a surgical endoscopy is performed, diagnostic endoscopy is included and cannot be coded separately.
  • Diagnostic endoscopy included in surgical endoscopy -- never code both
  • If endoscope does not reach the intended site, code the highest anatomical level reached
  • Multiple procedures during one endoscopy session may be coded with modifier -51 or add-on codes

Critical Coding Tips

Sinus Procedures

  • Nasal endoscopy codes (31231–31297) are unilateral -- use modifier -50 for bilateral
  • Sinusotomy codes specify the sinus: maxillary, frontal, ethmoid, sphenoid
  • Code each sinus separately when multiple sinuses are addressed

Bronchoscopy

  • Bronchoscopy codes (31622–31899) include the diagnostic procedure
  • Code additional procedures (biopsy, BAL, stent) separately using add-on codes
  • Rigid vs. flexible bronchoscopy = different code ranges

Thoracic Surgery

  • VATS (video-assisted thoracoscopic surgery) codes are separate from open thoracotomy codes
  • Chest tube insertion (32551) is frequently bundled -- check payer guidelines
  • Decortication codes differ by whether pleura is visceral or parietal

Lymph Nodes

  • Code lymph node procedures by approach (open vs. laparoscopic) and extent
  • Sentinel node biopsy uses both a surgical code and a nuclear medicine code
  • Lymphadenectomy: limited vs. radical = different codes

Unilateral vs. Bilateral Rules

Procedure TypeBilling Rule
Nasal endoscopy codesUnilateral -- use modifier -50 for bilateral
Sinus surgery codesUnilateral -- use modifier -50 for bilateral
Lung proceduresCheck code descriptor -- most specify one lung
Lymph node proceduresCode by laterality as documented

Common Exam Traps

  • Do not code diagnostic bronchoscopy when a surgical bronchoscopy is performed at the same session
  • Do not forget modifier -50 for bilateral nasal/sinus procedures
  • Do not confuse VATS codes with open thoracotomy codes
  • Do not code chest tube separately when included in the primary procedure
  • Do not miss add-on codes for additional procedures during bronchoscopy

CPC Exam Tips

  • Tab the endoscopy hierarchy note near the first bronchoscopy code
  • Write "UNILATERAL -- use -50 for bilateral" near nasal endoscopy codes
  • Know the difference between VATS and open procedures before exam day
