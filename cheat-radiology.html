Section Overview

The Radiology section (70010–79999) is fundamentally about understanding component coding. Most radiology services have a professional component (physician interpretation) and a technical component (equipment, supplies, technician). Mastering modifier -26 and -TC is essential for this section.

Core Concept:
No modifier = Global service (both professional and technical components)
Modifier -26 = Professional component only (physician reads/interprets the image)
Modifier -TC = Technical component only (equipment, supplies, technician)

Hospital outpatient departments bill ONLY -TC. Radiologists reading ONLY bill -26.

Critical Coding Tips

Contrast Rules

  • "Without contrast" = no IV contrast administered
  • "With contrast" = IV contrast administered
  • "Without, followed by with" = one code covers both phases (do NOT code separately)
  • Oral contrast alone does not constitute "with contrast" for CT purposes

X-Ray Rules

  • Code by number of views documented
  • Minimum views must be met for the code selected
  • Bilateral x-rays: check if code already includes bilateral or if modifier -50 is needed

Ultrasound Rules

  • Complete vs. limited study: code based on what was documented, not what was ordered
  • Complete study requires imaging of all required structures for that area
  • OB ultrasound: specify trimester (first, second/third) and number of fetuses
  • Transvaginal ultrasound may be coded in addition to transabdominal

Supervision & Interpretation (S&I)

  • S&I codes are always professional component only -- do not add modifier -26
  • Used when radiologist provides guidance for an interventional procedure
  • Surgeon codes the procedure; radiologist codes the S&I
  • Fluoroscopic guidance, ultrasound guidance, CT guidance = all S&I codes

Mammography

  • Screening mammography (77067): no symptoms, routine bilateral
  • Diagnostic mammography (77065–77066): performed due to symptoms or abnormal finding
  • Screening codes are already bilateral -- do not use modifier -50
  • Tomosynthesis (3D mammography) has separate add-on codes

Nuclear Medicine

  • Nuclear medicine codes include both the imaging and the radiopharmaceutical interpretation
  • Radiopharmaceutical supply may be coded separately (A9000s in HCPCS)
  • PET scan codes specify the radiopharmaceutical used

Common Exam Traps

  • Do not code "without" and "with" contrast separately when a "without followed by with" code exists
  • Do not add modifier -26 to S&I codes -- they are already professional component only
  • Do not use modifier -50 on screening mammography -- it is already bilateral
  • Do not code oral contrast as "with contrast" for CT purposes
  • Do not count individual images -- code by study, not by image count
  • Do not code incomplete studies without modifier -52
  • Do not confuse screening vs. diagnostic mammography -- indication matters

CPC Exam Tips

  • Write the -26/-TC/-global explanation at the very beginning of the Radiology section
  • Write "S&I = Professional only, no -26 needed" near the guidance codes
  • Tab: Diagnostic Radiology, Ultrasound, Nuclear Medicine, Radiation Oncology, Mammography
  • Know the contrast rules cold -- they appear in multiple question formats
  • Radiology is one of the most concept-driven sections. Understanding -26/-TC is half the battle.
